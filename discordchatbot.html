Discord Chatbot

Python/Flask/DB

This was a fun one! , while learning python I was practicing with a chatbot, and how do they work, I found an API that gives you a random inspirational quote, so I started thinking about how can I connect it with discords API, and here it is! My own inspirational chatbot.

The main obstacle was to keep my chatbot 24/7, I found out a website called uptime robot, the main purpose was to keep track of all your bots, I was lucky enough that it is completely free, so I connected with my project and voila! A free 24/7 chatbot!

I used python as the main programming language and also to import the APIs and libraries and hide my keys in an env file which was even easier thanks to replit.com integrations.
